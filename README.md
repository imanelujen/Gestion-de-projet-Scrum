## FRONTEND
├── React + Vite
├── Tailwind CSS
├── shadcn/ui (composants)
├── React Router (navigation)
├── Zustand (state)
├── Axios (HTTP)
├── react-beautiful-dnd (drag & drop)
├── Recharts (graphiques)
├── React Hook Form (formulaires)
└── date-fns (dates)

## BACKEND
├── Node.js + Express
├── Mysql
├── PostgreSQL (Supabase)


## Installation
cd backend && npm install
cd frontend && npm install

## Run
npm run dev
npm start
